Ls: list the files in a directory specified by the command-line argument.
Rm: used to remove or delete files and directories. "rm" stands for "remove."
Mv: used to move or rename files and directories. "mv" stands for "move."
Mkdir: make directory
Pwd: stands for "print working directory." used to display the current working directory.
Touch: used to create new empty files.
Chmod: used to change the permissions (mode) of files and directories.stands for "change mode."
Stat: refers to a system call or a function used to retrieve information about a file or file system
Wc:used to display information about files or standard input.stands for "word count"
Cat: used to display the contents of files
